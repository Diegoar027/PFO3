Para ejecutar el sistema:

1. Primero inicia el servidor:
python server.py
2 Luego puedes ejecutar el cliente en otra terminal:
python client.py

Este es un sistema básico que implementa:

Un servidor multihilo que acepta conexiones
Workers que procesan tareas en paralelo
Un cliente que puede enviar tareas y recibir resultados
Comunicación mediante sockets TCP
Serialización JSON para los mensajes
